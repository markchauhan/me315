# ME 315 — Machine Elements
**Northwestern University · McCormick School of Engineering**

Mechanical design course covering stress analysis, fatigue, gear systems, shafts, and bearings. Primary project: design of a three-shaft gear transmission system for an industrial power transmission application.

---

## Project — Three-Shaft Gear Transmission

### Specifications
| Parameter | Value |
|-----------|-------|
| Input speed | 2700 rpm |
| Output speed | 400 rpm |
| Speed ratio | 6.75:1 |
| Power input | 60 kW |
| Required reliability | 99% |
| Design life | 5 years |

### Design Scope
Each team member designed one complete shaft system including gears, shaft, and bearings:
- **Gear selection** — spur/helical gear pairs sized for the required ratio
- **Force analysis** — tangential, radial, and axial gear forces computed at each mesh
- **Shaft design** — diameter sizing for combined bending and torsional loading, stress concentration factors applied at keyways and shoulders
- **Fatigue analysis** — Goodman criterion for infinite life at 99% reliability
- **Bearing selection** — deep-groove ball bearings selected for L10 life at design load

### Design Priorities
- Structural compactness — minimized center distance
- Durability — conservative safety factors at fatigue-critical cross-sections
- Cost-effectiveness — standard catalog component selection where possible

### Tools
- SolidWorks for 3D modeling and assembly
- FEA for shaft deflection and stress validation
- Manual hand calculations for gear load, shaft stress, and bearing L10 life
